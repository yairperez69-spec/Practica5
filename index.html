¿Qué es HTML5?

HTML5 es el lenguaje para estructurar el contenido de las páginas web. Proporciona etiquetas semánticas y soporte para multimedia y gráficos.

¿Qué es CSS?

CSS define la apariencia del contenido HTML: colores, fuentes, tamaños, márgenes y diseño visual del sitio.

¿Qué es Java?

Java es un lenguaje de programación versátil que se utiliza para aplicaciones web, móviles y de escritorio, con gran portabilidad.
